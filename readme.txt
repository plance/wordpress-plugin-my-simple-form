== Description ==
The plugin

Using this plugin you can create form for aplication on job, purchase, donations, credit, etc. In the cpanel, you can configure your form fields, email, message, etc.

Используя данный плагин, вы сможете создать форму принятия анкет с сайта (устройство на работу, получение кредитов, покупка товаров, пр.). Через панель администратора можно настроить поля, задать им названия, устанавливать ограничения. Так же есть возможность настройки почтовых уведомлений.



== Installation ==
* Use standart WordPress instalator
* Or extract archive into "/wp-content/plugins/" folder
